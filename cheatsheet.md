All available commands:

GO: 
'go north' - moves the player to the room north of where the player currently is. Or if there is an unaccesable room/ no room, the player does not move. 

'go south' - moves the player to the room north of where the player currently is. Or if there is an unaccesable room/ no room, the player does not move.

'go east' - moves the player to the room north of where the player currently is. Or if there is an unaccesable room/ no room, the player does not move. 

'go west' - moves the player to the room north of where the player currently is. Or if there is an unaccesable room/ no room, the player does not move. 

DROP:
'drop fridgekey' - drops fridgekey from player inventory, if in inventory, in the room the player is currently in. 

'drop cagekey' - drops cagekey from player inventory, if in inventory, in the room the player is currently in. 

'drop towerkey' - drops towerkey from player inventory, if in inventory, in the room the player is currently in. 

'drop crowbar' - drops crowbar from player inventory, if in inventory, in the room the player is currently in. 


GRAB: 
'grab fridgekey' - adds fridgekey to player inventory, if not already in inventory and if key is in current room.

'grab cagekey' - adds cagekey to player inventory, if not already in inventory and if key is in current room. 

'grab towerkey' - adds towerkey to player inventory, if not already in inventory and if key is in current room. 

'grab crowbar' - adds crowbar to player inventory, if not already in inventory and if key is in current room.


OPEN: 
'open fridgedoor' - opens fridgedoor, if fridgedoor is in player's current room, allowing access to a new room and back.

'open cagedoor' - opens cagedoor, if cagedoor is in player's current room, allowing access to a new room and back.

'open towerdoor' - opens towerdoor, if towerdoor is in player's current room, allowing access to a new room and back.

'open manhole' - opens manhole, if manhole is in player's current room, allowing access to a new room and back.
